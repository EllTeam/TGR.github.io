# 🎮 The Gaming Race

The Gaming Race (TGR) is a web game by EllTeam.

## 🍺 Install EllToolsCLI

Install EllToolsCLI using Homebrew:

```bash
brew install elltools
```

## ⬇️ Install The Gaming Race

After installing EllToolsCLI:

```bash
elltools install tgr
```

or:

```bash
elltools install ellteam.github.io/EllGames/TGR
```

## 🌐 Play Online

You can also play The Gaming Race in your browser:

https://tgr.github.io

## 🏆 About

The Gaming Race is a competition where players test their gaming skills and try to get the highest score.
